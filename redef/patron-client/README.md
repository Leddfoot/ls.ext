#Patron client (node.js).
===

Patron client is a viewer application for data from [Services](https://github.com/digibib/ls.ext/blob/master/redef/services/README.md).

Technologies in use:

* Node.JS
* HTML
* Server
  * http
  * Express
* Templating
  * Handlebars
* Test
  * Chai
  * Cheerio
  * Router

See [Makefile](Makefile).
